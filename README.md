# myReceipts 
AWS로 만드는 myReceipts 프로젝트 따라하기 설명서. 
myReceipts는 이메일로 발송 되는 카드결제 내역을 Google Spreadsheet로 수집하는 간단한 서비스입니다. 도메인 등록에서 부터, AWS에서 서비스 운영까지의 과정을 step-by-step로 설명합니다.
